# 03 — 攻略ナレッジ読み込み

## 目的

`knowledge/dq3-fc/` 内の `.md` を**全件**読み込み、決定的な順序で1つの文字列に連結し、メモリにキャッシュする関数を実装する。

## 背景

Claude API は URL を開かない。「AI に認識させる」＝テキストをプロンプトに含めること。リトリーバルは作らず、全ファイル連結で渡す。現状 `knowledge/dq3-fc/` に 45 ファイル（`_ai-notes.md` 含む）。

## 対象ファイル

- `src/lib/knowledge.ts`

## 実装の要点（CLAUDE.md「Core architecture 2」/ REQUIREMENTS.md §8）

- 読み込み元は `path.join(process.cwd(), 'knowledge', 'dq3-fc')`。**サーバ側専用**（`public/` でも `src/` でもない）。
- `.md` を全件読む。**ファイル名で決定的にソート**（`readdir` の順序は OS 依存）。
- **`_ai-notes.md` を先頭に固定**。残りはソート順。
- 連結結果はモジュールスコープ変数に**キャッシュ**し、リクエスト毎に読み直さない。
- 順序が毎回同じであることがプロンプトキャッシュヒットの前提。
